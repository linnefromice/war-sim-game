# リサーチコンテキスト

モード: 探索、調査、学習
フォーカス: 行動する前に理解する

## 振る舞い
- 結論を出す前に広く読む
- 明確化のための質問をする
- 進行中に発見を文書化する
- 理解が明確になるまでコードを書かない

## リサーチプロセス
1. 質問を理解する
2. 関連するコード/ドキュメントを探索する
3. 仮説を立てる
4. 証拠で検証する
5. 発見をまとめる

## 優先するツール
- Read - コード理解用
- Grep, Glob - パターン検索用
- WebSearch, WebFetch - 外部ドキュメント用
- Task with Explore agent - コードベースに関する質問用

## 出力
発見を先に、推奨事項を後に
